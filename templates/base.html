Hitcounter Index Page

To use the hitcounter, simply add this img tag to your site:

<a href="{{url_for('index', _external=True)}}">
    <img src="{{ url_for('UniqueTicker', _external=True) }}" alt="A hitcounter in the style of a seven-segment display." />
</a>


Or, if you only want to track all hits, use this instead:

<a href="{{url_for('index', _external=True)}}">
    <img src="{{ url_for('CurrentTicker', _external=True) }}" alt="A hitcounter in the style of a seven-segment display." />
</a>
